📂 File Organizer (Python Automation Tool)

A Python-based automation tool that organizes files into categorized folders such as Images, Documents, Videos, and more for efficient file management.

---

🚀 Features

* Automatically sorts files based on their type
* Creates folders if they don’t exist
* Supports multiple file categories (Images, Documents, Videos, Music, Others)
* Easy to use and beginner-friendly

---

🛠️ Tech Used

* Python
* os module
* shutil module

---

📂 How It Works

* The user provides a folder path
* The script scans all files in the folder
* Files are moved into respective folders based on their extensions

---

▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run the script:

python file_organizer.py

4. Enter the folder path when prompted (e.g., Downloads folder)

---

 ⚠️ Important Note

* Do not run on important folders without backup
* Test using a sample folder first

---

📌 Example

Before:

Downloads/
  file1.jpg
  file2.pdf
  video.mp4

After running:

Downloads/
  Images/file1.jpg
  Documents/file2.pdf
  Videos/video.mp4
